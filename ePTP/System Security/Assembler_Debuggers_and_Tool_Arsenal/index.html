2_Assembler_Debuggers_and_Tool_Arsenal

  • Assembly is a Mnemonic code language, also known as opcode (operation code).
  • Types of Assembler
    1. MASM → x86 architecture assembler, use the intel syntax for Microsoft windows, MS-DOS.
    1. NASM → x86 architecture assembler, use to write 16-bit, 32-bit, and 64-bit (x86-64), popular for Linux.
    1. GNU Assembler (GAS) → used for the GNU project.
    1. Flat Assembler → x86, supports Intel style assembly language on the IA-32 and x86-64.
  • Assembler does some important operations like assigning memory location to variables and instructions.
  • What a linker does is take one or more object files and combine them to create the executable file.
  • An example of these object files are the kernel32.dll and user32.dll which are required to create a windows executable that accesses certain libraries.
  • High level function such as strcpy(), consists of multiple ASM instructions put together.
  • Each opcode represented in one line and contains the instruction (MOV, PUSH, etc) and the operands.
    • No of operands differ from one instruction to another.
  • Q (quad 64 bits), L(long 32 bits), W (word 16 bits), B (byte 8 bits) → AT&T (linux)
    • Instruction → Source → Destination.
  • Subroutines are implemented by using the CALL and RET instruction pair.
  • CALLfunction pushes the EIP to the stack, and RET popes the last element of the stack.
  • MinGW64 (Minimalist GNU compiler for Windows) → folder has all compiling tools.
  • Debugger (Immunity Debugger)
    1. Disassembler panel:
      1. first column → address
      1. Second column → machine code
      1. Third column → assembly code
      1. Fourth column → debugger comment
    1. Register panel → contain name of registers, and their content.
    1. The Memory Dump panel
    1. Stack Panel → Show the current stack thread
      1. first column → address
      1. Second column → content in that address
      1. Third column → debugger comment

  • objdump.exe → is used to disassemble executable files.
    • Command → objdump -d -Mintel [.exe file] > [asm.txt]
