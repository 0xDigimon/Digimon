5_Cryptography_and_Password_Cracking

  • Cryptography has several types:
    • Based on the keys are used
      1. Symmetric key cryptography → DES(Data Encryption Standard), 3DES, AES(Advanced Encryption Standard), RC4, Blowfish.
      1. Public key cryptography(Asymmetric)
        • The whole concept of public key cryptography is based on the factorization mathematical problem.
          • Factorization problem : There is no known efficient method to find all the factors of any big number (say 300 digits or more).
    • Based on the method of handling data
      1. Block Cipher → handle data with blocks (8 bytes, 16bytes), and it has a lot of modes.
        1. ECB (Electronic Code Book) → the message is divided into blocks and each block is encrypted separately.
        1. CBC (Cipher Block Chaining)
      1. Stream Cipher → handle data with one byte in a time.
    • Examples
      • Caesar’s cipher → It works by replacing each letter in the plain text by a letter some fixed number of positions down the alphabet.
  • Avalanche → means that any change in the message/input, will provide a different output.
  • The goal of PKI → is to make sure that the identity of invidiual or orgnization is effectivly certificated and verified.
  • X.509 → is the standard for public key certificates.
    • X.509 certificates are widely used in protocols like SSL/TLS, SET, S/MIME, IPsec and more.
  • PGP
    • PGP (Pretty Good Privacy) → copmuter program (windows) is used to encrypt files, provdie digital signature.
      • PGP encryption uses public key cryptography, and binds the public key with an e-mail.
    • OpenPGP is a set of standards which describes the formats for encrypted messages, keys, and digital signatures.
    • The PGP software puts them (Public keys of your friend) in a file, called your keyring .
    • To send message
      1. PGP generates a symetric key and then encrypt it with a public key
      1. encrypt the message with a symmetric key
      1. Send the message to the address related to the public key
    • PGP encrypts only 160-bits of message.
  • SSH
    • is used for exchange data between two devices, secure file transfer, port forwarding.
    • SSH tunnels are used to transfer unecrypted data over the network through encrypted tunnels.
    • Pivoting→ tunneling traffic between two networks, through a machine under attacker’s control.
    • Types of SSH tunneling
      1. local port forwarding
        • Here the data comes to the client SSH, fowarding to the SSH server, then to the destination.
      1. Remote

Crypto Attacks
  • Cryptographic attacks are attempts to subvert the security of the crypto algorithms by exploiting weaknesses with the goal to decipher ciphertext without knowing the key.
  • The explored key space is far smaller compared to the Brute force attack. Considering a 4 letters, lowercase key a dictionary might attempt: - abba , big, bob, bed, dig, dog, cat, doll, duff, While a brute force would attempt: - aaaa , aaab, aaac, aaad,
  • 4. side channel attacks
    • Side channel attacks don't rely just on plaintext/ciphertext information to attack crypto algorithms.
      • They also take into account physical implementation including the hardware used to encrypt or decrypt data.
      • Time taken to perform an encryption, CPU cycles used and even absorbed power variation during the algorithm can produce important information to a crypto analyst.

Windows Passwords
  • SAM (Security Account Manager) → is a database that stores passwords.
    • C:\Windows\System32\config
      • But, it is not accessible while the operating system is running. (except SYSTEM prev).
    • These values are also stored in the registry at: HKEY_LOCAL_MACHINE\SAM
  • it stores the passwords in two formats:
    • LM hash
    • NT hash
  • 1. Stealing the hashes

      • Here you need physical access to the machine
